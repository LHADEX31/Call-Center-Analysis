# Call-Center- Performance-Analysis
Data Analysis using Power BI
#Project Overview
This project focuses on analyzing call-center performance data to understand how efficiently customer inquiries are handle and how service quality impacts customers satisfaction.  The goal was not just to report numbers , but to uncover patterns that can help to improve operational efficiency and customer experience.

## Objectives
- Evaluate key call center performance metrics
- Identify factors affecting customers satisfaction
- Monitor agent performance and services responsiveness
- provide data-driven recommendations for improvement

## Key Metrics Analyzed
- Average Call Duration
- Response Time and SLA Compliance
- Customer Satisfaction Score (CSAT)
- Call Volume Trends
- Agent Performance Metrics
 
  
## Tools Used
- Microsoft Excel (data xleaning and preliminary analysis)
- Power BI (data modeling, DAX ,and interactive dashboards)

## Key Insights
- Longer response times were strongly associated with lower customer satisfaction scores
- Peak call periods showed increased SLA breaches, indicating capacity and staffing challenges
- Certaain agents consistently handle calls more efficiently while maintaining higher CSAT suggesting best practices that can be replicated.

  
## Recomendations
- Optimize staffing during peak hours to reduce response time and SLA violations
- Use performance benchmarks from top-performance agents to guide training and coaching
- Continuously monitor KPIs through dashboards to ssupport real-time decision making.


## Outcome
  The analysis provide a clear view of call center performance and highlights actionable steps to improved service quality , operational efficiency, and customer satisfactions. This project demostrate the practical use of dtata analysis to support business decisions in a customer dacing enviroment.
